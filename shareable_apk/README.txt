MindFlow - Shareable APK
========================

Requirements:
- Android 8.0 (API 26) or higher.

To use/share this APK:
1. In the sidebar file explorer of Google AI Studio, locate the "shareable_apk" folder.
   - Note: Because default IDE file explorers often filter/hide binary files like ".apk", you may only see "MindFlow_apk.zip" and "README.txt" in that folder.
2. Select "MindFlow_apk.zip" from the file tree, click the three dots next to it, and download it to your computer/phone.
3. Once downloaded, rename the file extension from "MindFlow_apk.zip" back to "MindFlow.apk" (do NOT unzip/extract it; simply rename ".zip" to ".apk").
4. Transfer the ".apk" file to your Android device.
5. Open the APK file on your device to install MindFlow.

(Note: You may need to enable "Install from unknown sources" in your device's security settings if prompted.)
